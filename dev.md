## 安装 Docker

```
dnf config-manager --add-repo="https://mirrors.cloud.tencent.com/docker-ce/linux/centos/docker-ce.repo"

dnf list docker-ce

dnf install -y docker-ce --nobest

systemctl start docker

docker info
```
